# The `utils/` Directory

This directory is designed to hold different programs for interpreting
Bergamot's wire protocol. Likewise, any of the programs in this directory can
be considered to be Bergamot.

Bergamot utilities should obey a handful of rules, which are ~~written in
detail in `utils/GUIDELINES.md`.~~

IN THIS DIRECTORY
---

#### `bergamot-dump`

A simple commandline utility to display/dump Bergamot data to disk.

#### `bergamot-overseer`

A Flask server for viewing and filtering Bergamot data from a web interface.
