Getting Started

To gather data to analyze, first connect an Agent instance to this machine. The Overseer is currently expecting a connection at port JAVASCRIPT REQUIRED. When an agent connects, it will automatically start populating the database and you can see how long it's connected at the top of the screen. Analysis can be done while the Agent is still transmitting data, but it is worth noting that this program is not yet capable of live analysis, and can only sniff retroactively.
